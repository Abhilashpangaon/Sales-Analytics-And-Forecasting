# Sales Analytics and Forecasting System

## Overview
An end-to-end data analytics project using SQL, Python, Machine Learning, and Power BI.

## Tech Stack
- SQL
- Python (Pandas, ML)
- Power BI

## Key Features
- Data cleaning & transformation
- KPI analysis
- Predictive sales forecasting
- Interactive dashboard

## Business Impact
Helps management make data-driven decisions on sales, inventory, and expansion.
